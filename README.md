# PropManage - Automated Property Management & Booking System

## Goal
Prove the ability to handle complex business logic, data processing, and stability using robust frameworks.

## Tech Stack
- **Frontend:** Vue.js
- **Backend:** Python Django
- **Database:** PostgreSQL
- **Async Tasks:** Celery + Redis
- **DevOps:** Docker, Staging/Production environments

## Key Features
- **Complex Scheduling:** Booking algorithms, timezone handling.
- **Background Jobs:** Automated invoicing, email notifications.
- **Reporting:** Data visualization with Pandas/Matplotlib.
- **RBAC:** Distinct roles for Admin, Landlord, and Tenant.
